# Personal Finance Manager (PFM)

Система учета и анализа личных финансов.

## Описание

Personal Finance Manager - это desktop приложение для ведения учета личных финансов. Позволяет добавлять доходы и расходы, категоризировать их, просматривать баланс и формировать простые отчеты.

## Функционал

- Добавление, редактирование и удаление финансовых операций
- Управление категориями операций
- Расчет текущего баланса
- Фильтрация операций по дате и категории
- Экспорт данных в CSV

## Технологии

- Python 3.8+
- Tkinter (GUI)
- SQLite (база данных)
- SQLAlchemy (ORM)

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python main.py
```

## Структура проекта

- `src/` - исходный код
- `tests/` - тесты
- `docs/` - документация
- `resources/` - ресурсы приложения
