## Задание №1: Классы эквивалентности для калькулятора

**Требования:**
- В калькулятор можно вводить только цифры
- Калькулятор поддерживает ввод цифр от 0 до 9999

**Классы эквивалентности:**

1. **Допустимый класс:** Числа от 0 до 9999
2. **Недопустимый класс 1:** Отрицательные числа (от -∞ до -1)
3. **Недопустимый класс 2:** Числа больше допустимого диапазона (от 10000 до +∞)
4. **Недопустимый класс 3:** Нецифровые символы (буквы, спецсимволы)

**Граничные значения:**
- Минимальное граничное: 0
- Перед минимальным: -1
- После минимального: 1
- Максимальное граничное: 9999
- Перед максимальным: 9998
- После максимального: 10000

## Задание №3: Классы эквивалентности для загрузки картинки

**Требования:**
- Картинку можно загрузить размером от 0 до 3МБ
- В негативных классах можно учесть форматы, не относящиеся к картинкам

**Классы эквивалентности:**

**Допустимые:**
1. Размер файла: 0 < размер ≤ 3MB
2. Форматы изображений: JPG, JPEG, PNG, GIF, BMP, WEBP, SVG

**Недопустимые:**
3. Размер файла: размер = 0 байт (пустой файл)
4. Размер файла: размер > 3MB
5. Некорректные форматы: PDF, DOC, TXT, MP4, ZIP, EXE и другие не-изображения
6. Отсутствие файла (не выбран файл)
7. Поврежденный файл изображения

**Граничные значения:**
- 0 байт
- 1 байт
- 3MB - 1 байт
- Ровно 3MB (3145728 байт)
- 3MB + 1 байт
